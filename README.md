# DazLookAtScript
Look At script for Daz Studio 6

Point a figure's eyes, head, and torso at any target.

## How to Install and Run

1. Copy the script file to your Daz Studio library scripts folder.
2. Open Daz Studio, locate the script in your Content Library, and double-click it to run or you can right click it to create a custom action to open it from your toolbar scripts/favorites menu.
